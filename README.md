# Amazon Prime EDA Project

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on Amazon Prime Movies and TV Shows dataset to understand content trends, audience preferences, and performance factors.

## 📊 Key Features
- Data Cleaning & Preprocessing
- Univariate, Bivariate, Multivariate Analysis
- 20+ Visualizations
- Business Insights & Recommendations

## 📁 Files Included
- real eda.ipynb → Main notebook
- titles.csv → Titles dataset
- credits.csv → Credits dataset

## 🎯 Objective
To analyze content performance and support better content strategy decisions.

## 🚀 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Outcome
The analysis helps understand what type of content performs well and how platforms can improve engagement.
